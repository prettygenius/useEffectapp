## Features

-  Use the useEffect hook for understanding concepts such as function cleanup
-  demo uses cleanup function when component unmounts to prevent unnecessary memory usage

## Technologies Used

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)


## Getting Started

1. **Clone the repository:**

```bash
git clone https://github.com/prettygenius/useEffectapp.git
cd useEffect-app
```
Install dependencies:
```
npm install
```
``` 
npm run dev
```
